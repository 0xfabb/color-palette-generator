# Design Contribution Guidelines

## Introduction

If you are reading this, I guess you are interested in contributing to the design of the Color Conjure. First of all, thank you for your interest and time. This document will guide you through the process and requirements for contributing to the design of the project.

## Contribution Guidelines

1. **Figma Design** - You can create a design in Figma and share the link with us. Please update this document with the Date, Title of the design (like which feature), the Figma link and a screenshot of the design, which will be saved in this same directory.

2. **Excalidraw** - You can create a design in Excalidraw and share the link with us. Please update this document with the Date, Title of the design (like which feature), the Excalidraw link and a screenshot of the design, which will be saved in this same directory.

3. Please note that version number of the png file should be updated with each new design. For example, if the last design was `v01_design.png`, then the next design will be `v02_design.png`.

4. To ensure the size of the images is not too large, please add the images in the `.png` format and compress them before adding them to the repository. Use [TinyPNG](https://tinypng.com/) to compress the images.

## Raise a PR

Once you have created a design and updated this document, please raise a PR with the title `Design Contribution: [Feature Name]`. The maintainers will review the design and merge it if it meets the requirements.

Read the [CONTRIBUTING.md](../CONTRIBUTING.md) file for more information on how to contribute to the project.
